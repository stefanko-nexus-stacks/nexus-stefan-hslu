## Draw.io

![Draw.io](https://img.shields.io/badge/Draw.io-F08705?logo=diagramsdotnet&logoColor=white)

**Flowchart and diagramming tool for creating technical diagrams**

Professional diagramming tool (also known as diagrams.net) for creating flowcharts, UML diagrams, network diagrams, and technical documentation. Features include:
- Flowcharts, UML, ERD, network diagrams
- Extensive shape libraries
- Import/export to various formats (XML, PNG, SVG, PDF)
- No account required
- Offline-capable

| Setting | Value |
|---------|-------|
| Default Port | `8083` |
| Suggested Subdomain | `drawio` |
| Public Access | No (protected by Cloudflare Access) |
| Website | [diagrams.net](https://www.diagrams.net) |
| Source | [GitHub](https://github.com/jgraph/drawio) |
